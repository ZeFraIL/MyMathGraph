# Описание класса SinusCosinus

## 1. Общая информация
*   **Имя класса:** `SinusCosinus`
*   **Тип:** Activity
*   **Назначение:** Этот класс отображает тригонометрические функции: синус, косинус и синус с фазовым сдвигом. Он показывает, как эти волны соотносятся друг с другом.
*   **Взаимодействие:** Запускается из `MainActivity`. Использует `GraphView` с несколькими сериями данных.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `amp` | `int` | Амплитуда (высота волны) | `onCreate` |
| `numPoints` | `int` | Разрешение (количество точек для 720 градусов) | `onCreate` |
| `graphSiCo` | `GraphView` | Контейнер для графика | `onCreate` |
| `y` | `double` | Вычисленное значение Y | `onCreate` |
| `f` | `double` | Константа фазового сдвига ($\pi/4$) | `onCreate` |
| `series1`, `series2`, `series3` | `LineGraphSeries` | Разные линии для Sin, Cos и сдвинутого Sin | `onCreate` |

## 3. Методы класса

### Имя метода: `onCreate`
*   **Тип метода:** `protected`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `Bundle savedInstanceState`
*   **Что делает метод:**
    1.  Настраивает интерфейс.
    2.  Вычисляет и заполняет `series1` (синус) черным цветом.
    3.  Вычисляет и заполняет `series3` (синус + фазовый сдвиг) красным цветом.
    4.  Вычисляет и заполняет `series2` (косинус) пурпурным цветом.
    5.  Добавляет все серии на график.
*   **Когда вызывается:** При запуске Activity.
*   **Что важно понимать:** Цикл идет от 0 до 720, представляя градусы. Формула `i * Math.PI / 180` переводит градусы в радианы, что необходимо для функций `Math.sin()`.

## 4. Жизненный цикл
*   **`onCreate()`:** Вся логика рисования происходит здесь сразу при создании.

## 5. Взаимодействие с интерфейсом (UI)
*   Отображает график напрямую без ввода пользователя в этом конкретном классе (параметры заданы в коде).

## 6. Взаимодействие с другими компонентами
*   Стандартный переход из `MainActivity`.

## 7. Общая логика класса
Он заранее вычисляет три разные волны и накладывает их на одну систему координат, чтобы показать их различия (фазу и форму).

## 8. Упрощенное объяснение
Представьте трех танцоров, движущихся волнами. Один начинает с нуля (синус), другой — сверху (косинус), а третий — чуть раньше (сдвинутый синус). Этот класс рисует пути всех трех танцоров одновременно, чтобы вы могли их сравнить.
