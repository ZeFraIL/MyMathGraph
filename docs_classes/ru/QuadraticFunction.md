# Описание класса QuadraticFunction

## 1. Общая информация
*   **Имя класса:** `QuadraticFunction`
*   **Тип:** Activity (Экран приложения)
*   **Назначение:** Этот класс визуализирует квадратичную функцию (параболу) вида $y = ax^2 + bx + c$. Он вычисляет форму параболы на основе коэффициентов пользователя и позволяет интерактивно исследовать точки.
*   **Взаимодействие:** Запускается из `MainActivity`. Использует `GraphView` для отрисовки и `Toast` для отображения координат точек.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `etA2`, `etB2`, `etC2` | `EditText` | Поля ввода для $a, b, c$ | `onCreate`, `viewGraph` |
| `stA`, `stB`, `stC` | `String` | Строки для чтения из EditText | `viewGraph` |
| `a`, `b`, `c` | `int` | Числовые коэффициенты | `viewGraph` |
| `numPoints` | `int` | Количество сегментов для рисования кривой | `viewGraph` |
| `graphQ` | `GraphView` | Контейнер для графика | `onCreate`, `viewGraph` |
| `d` | `double` | Дискриминант ($\Delta = b^2 - 4ac$) | `viewGraph` |
| `x1`, `x2` | `double` | Корни уравнения | `viewGraph` |
| `xmin` | `double` | X-координата вершины | `viewGraph` |
| `dx` | `double` | Размер шага между точками | `viewGraph` |
| `xL`, `xR` | `double` | Левая и правая границы для рисования | `viewGraph` |

## 3. Методы класса

### Имя метода: `onCreate`
*   **Тип метода:** `protected`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `Bundle savedInstanceState`
*   **Что делает метод:** Настраивает интерфейс и инициализирует ссылки на поля ввода и представление графика.

### Имя метода: `viewGraph`
*   **Тип метода:** `public`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `View view`
*   **Что делает метод:**
    1.  Считывает и анализирует $a, b, c$.
    2.  Вычисляет дискриминант `d`.
    3.  Определяет диапазон просмотра (`xL` - `xR`) в зависимости от наличия корней и направления параболы (вверх/вниз).
    4.  Вычисляет шаг `dx` на основе `numPoints`.
    5.  Запускает цикл от 1 до `numPoints+1`, вычисляя $y$ для каждого $x$ и добавляя его в серию данных.
    6.  Включает анимацию и добавляет слушатель для отображения сообщения `Toast` с координатами при нажатии пользователем на точку.
*   **Когда вызывается:** При нажатии на кнопку.
*   **Что важно понимать:**
    *   Логика определения `xL` и `xR` гарантирует, что «интересная» часть параболы (вершина и корни) будет видна.
    *   `series.setAnimated(true)` делает процесс рисования плавным.

## 4. Жизненный цикл
*   **`onCreate()`:** Инициализация Activity.

## 5. Взаимодействие с интерфейсом (UI)
*   Использует три `EditText` для ввода.
*   Сообщения `Toast` используются для интерактивной обратной связи.

## 6. Взаимодействие с другими компонентами
*   Стандартный переход из `MainActivity`.

## 7. Общая логика класса
Это математический калькулятор, который переводит алгебраическую формулу в набор из 100 небольших отрезков линий, которые выглядят как плавная кривая (парабола).

## 8. Упрощенное объяснение
Думайте об этом как о «Строителе парабол». Вы даете ему рецепт ($a, b, c$), и он вычисляет 100 крошечных точек и соединяет их. Если вы коснетесь любой точки, он точно скажет вам, где она находится.
