# Описание класса LinearFunction

## 1. Общая информация
*   **Имя класса:** `LinearFunction`
*   **Тип:** Activity (Экран приложения)
*   **Назначение:** Этот класс отвечает за визуализацию линейной функции вида $y = ax + b$. Он принимает от пользователя коэффициенты 'a' и 'b' и рисует соответствующую прямую на графике.
*   **Взаимодействие:** Запускается из `MainActivity` через `Intent`. Использует библиотеку `GraphView` для отрисовки математического графика.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `etA1` | `EditText` | Поле ввода для коэффициента 'a' | `onCreate`, `viewGraph` |
| `etB1` | `EditText` | Поле ввода для коэффициента 'b' | `onCreate`, `viewGraph` |
| `stA` | `String` | Временное хранение текста из `etA1` | `viewGraph` |
| `stB` | `String` | Временное хранение текста из `etB1` | `viewGraph` |
| `a` | `int` | Числовое значение коэффициента 'a' | `viewGraph` |
| `b` | `int` | Числовое значение коэффициента 'b' | `viewGraph` |
| `graphL` | `GraphView` | Компонент интерфейса для отображения графика | `onCreate`, `viewGraph` |
| `x` | `double` | Координата x корня функции ($y=0$) | `viewGraph` |
| `xL` | `double` | Координата x для левой точки прямой | `viewGraph` |
| `xR` | `double` | Координата x для правой точки прямой | `viewGraph` |
| `y` | `double` | Вычисленная координата y | `viewGraph` |

## 3. Методы класса

### Имя метода: `onCreate`
*   **Тип метода:** `protected`
*   **Возвращаемое значение:** `void` (ничего не возвращает)
*   **Параметры:** `Bundle savedInstanceState` (состояние экрана)
*   **Что делает метод:**
    1.  Вызывает конструктор родительского класса.
    2.  Устанавливает макет интерфейса из файла `activity_linear_function`.
    3.  Связывает Java-переменные (`etA1`, `etB1`, `graphL`) с элементами в XML-файле с помощью `findViewById`.
*   **Когда вызывается:** Автоматически системой Android при создании экрана.

### Имя метода: `viewGraph`
*   **Тип метода:** `public`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `View view` (кнопка, на которую нажали)
*   **Что делает метод:**
    1.  Считывает текст из полей ввода.
    2.  Проверяет, не пусты ли поля.
    3.  Преобразует текст в числа (`Integer.parseInt`).
    4.  Вычисляет корень функции: $x = -b / a$.
    5.  Создает новую серию данных для графика (`LineGraphSeries`).
    6.  Вычисляет две точки: одну на 5 единиц левее корня, другую на 5 единиц правее.
    7.  Добавляет эти точки в серию и отображает на графике.
*   **Когда вызывается:** При нажатии пользователем на кнопку "View Graph".
*   **Что важно понимать:**
    *   **Возможная ошибка:** Если пользователь введет `a = 0`, программа аварийно завершится из-за деления на ноль.
    *   **Особенность:** Используются только две точки, так как линейная функция — это всегда прямая.

## 4. Жизненный цикл
*   **`onCreate()`:** Вызывается при запуске Activity. Здесь мы настраиваем интерфейс и находим визуальные элементы.

## 5. Взаимодействие с интерфейсом (UI)
*   **Элементы:** `EditText` (для ввода чисел), `Button` (для запуска расчета), `GraphView` (для вывода результата).
*   **Связь:** `findViewById` связывает ID из XML с объектами в коде.
*   **События:** Обработка нажатия на кнопку (задано в XML через `android:onClick="viewGraph"`).

## 6. Взаимодействие с другими компонентами
*   **Intent:** Это Activity открывается из `MainActivity`.
*   **Передача данных:** Данные между экранами не передаются; всё зависит от ввода пользователя.

## 7. Общая логика класса
1.  Пользователь вводит 'a' и 'b'.
2.  Нажимает кнопку.
3.  Приложение находит точку пересечения с осью X.
4.  Приложение проводит через эту точку прямую линию.

## 8. Упрощенное объяснение
Представь, что у тебя есть линейка и ты хочешь нарисовать линию на бумаге. Для этого тебе нужны как минимум две точки. Этот класс вычисляет эти две точки на основе введенных чисел и использует «цифровую линейку» (библиотеку `GraphView`), чтобы нарисовать линию за тебя.

**Аналогия:** Это как рецепт. Ты даешь ингредиенты (числа 'a' и 'b'), а программа «готовит» из них рисунок линии.
