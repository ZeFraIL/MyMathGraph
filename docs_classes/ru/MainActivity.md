# Описание класса MainActivity

## 1. Общая информация
*   **Имя класса:** `MainActivity`
*   **Тип:** Activity (Главный экран)
*   **Назначение:** Это точка входа в приложение. Оно служит главным меню, где пользователи могут выбрать, какую математическую функцию они хотят изучить.
*   **Взаимодействие:** Запускает другие экраны (`LinearFunction`, `QuadraticFunction` и т. д.) в зависимости от выбора пользователя.

## 2. Переменные (поля класса)
*В теле класса не определены явные переменные-члены; используются только локальные переменные внутри методов для переходов.*

## 3. Методы класса

### Имя метода: `onCreate`
*   **Тип метода:** `protected`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `Bundle savedInstanceState`
*   **Что делает метод:** Инициализирует Activity и устанавливает макет из файла `activity_main.xml`.
*   **Когда вызывается:** При запуске приложения.

### Методы навигации (`goLinear`, `goQuadratic`, `goSeveral`, `goSinCos`)
*   **Тип метода:** `public`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `View view`
*   **Что делают:** Создают `Intent` (намерение) для перехода от `MainActivity` к целевому экрану и вызывают `startActivity(go)`.
*   **Когда вызываются:** При нажатии соответствующей кнопки в главном меню.

## 4. Жизненный цикл
*   **`onCreate()`:** Настраивает макет главного меню.

## 5. Взаимодействие с интерфейсом (UI)
*   **Элементы:** Несколько кнопок (`Button`).
*   **Связь:** Реализована через атрибут `android:onClick` в XML-файле макета.

## 6. Взаимодействие с другими компонентами
*   **Intents:** Использует явные интенты для запуска других экранов.

## 7. Общая логика класса
Класс работает как «роутер» или распределитель. Он сам не выполняет вычислений, а просто направляет пользователя в нужный раздел приложения.

## 8. Упрощенное объяснение
Представьте, что `MainActivity` — это «Входная дверь» или «Главное меню» в ресторане. Вы не едите прямо у двери; вы используете меню, чтобы выбрать, куда вы хотите пойти (Линейные функции, Квадратичные и т. д.).
