# Описание класса CircleFunction

## 1. Общая информация
*   **Имя класса:** `CircleFunction`
*   **Тип:** Activity
*   **Назначение:** Этот класс визуализирует окружность и радиус. Поскольку окружность не является функцией в строгом смысле (одному $x$ могут соответствовать два $y$), она рисуется с использованием двух отдельных математических серий: верхней и нижней половин.
*   **Взаимодействие:** Запускается из `MainActivity`.

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `amp` | `int` | Радиус окружности | `onCreate` |
| `numPoints` | `int` | Количество точек для расчета половин | `onCreate` |
| `graphCircle` | `GraphView` | Компонент интерфейса для отображения окружности | `onCreate` |
| `series1`, `series2` | `LineGraphSeries` | Используются для верхней и нижней половин окружности | `onCreate` |
| `series3` | `LineGraphSeries` | Используется для рисования радиуса под углом 30 градусов | `onCreate` |

## 3. Методы класса

### Имя метода: `onCreate`
*   **Тип метода:** `protected`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `Bundle savedInstanceState`
*   **Что делает метод:** 
    1. Инициализирует интерфейс.
    2. **Верхняя половина:** Цикл от -100 до 100, расчет $y = \sqrt{R^2 - x^2}$ (где $R$ — это `amp`).
    3. **Нижняя половина:** Цикл от -100 до 100, расчет $y = -\sqrt{R^2 - x^2}$.
    4. **Радиус:** Добавляет две точки: (0,0) и точку на окружности под углом 30 градусов, используя $x = R \cdot \cos(30^\circ)$ и $y = R \cdot \sin(30^\circ)$.
    5. Добавляет все три серии на график.
*   **Когда вызывается:** При создании Activity.

## 4. Жизненный цикл
*   **`onCreate()`:** Все вычисления и рисование происходят сразу при открытии.

## 5. Взаимодействие с интерфейсом (UI)
*   **GraphView:** Отображает статичный рисунок окружности и её радиуса.

## 6. Взаимодействие с другими компонентами
*   Стандартный переход через Intent.

## 7. Общая логика класса
Поскольку стандартные графики обычно ожидают один $y$ для каждого $x$, приложение «обманывает» график, рисуя две отдельные линии (одну для верха и одну для низа), которые соединяются по бокам, образуя полную окружность. Также добавляется декоративный радиус для демонстрации конкретного угла.

## 8. Упрощенное объяснение
Рисование окружности на графике похоже на рисование двух улыбок: одной обычной для низа и одной перевернутой для верха. Когда они идеально сходятся, получается круг. Приложение также рисует красную «стрелку» из центра к краю, чтобы показать радиус.

**Аналогия:** Как сложить две половинки апельсина, чтобы получился целый апельсин.
