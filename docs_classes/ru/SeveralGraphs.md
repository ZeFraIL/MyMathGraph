# Описание класса SeveralGraphs

## 1. Общая информация
*   **Имя класса:** `SeveralGraphs`
*   **Тип:** Activity
*   **Назначение:** Этот класс демонстрирует, как наложить несколько квадратичных функций (парабол) на один график. Он последовательно переключается между четырьмя предопределенными наборами коэффициентов, чтобы показать, как изменение 'a', 'b' или 'c' влияет на форму и положение параболы.
*   **Взаимодействие:** Запускается из `MainActivity`. Использует общий `GraphView` для отображения нескольких серий данных (`LineGraphSeries`).

## 2. Переменные (поля класса)
| Имя | Тип | Назначение | Где используется |
| :--- | :--- | :--- | :--- |
| `a`, `b`, `c` | `int` | Базовые коэффициенты для парабол | `viewGraph`, `buildGraph` |
| `numPoints` | `int` | Количество точек для расчета каждого графика | `buildGraph` |
| `i` | `int` | Счетчик для отслеживания того, какой график рисовать следующим | `viewGraph`, `buildGraph` |
| `graphS` | `GraphView` | Компонент интерфейса для отображения графиков | `onCreate`, `buildGraph` |
| `dx` | `double` | Размер шага для расчета по оси X | `buildGraph` |
| `colors` | `int[]` | Массив цветов (черный, синий, красный, пурпурный) для графиков | `buildGraph` |

## 3. Методы класса

### Имя метода: `onCreate`
*   **Тип метода:** `protected`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `Bundle savedInstanceState`
*   **Что делает метод:** Настраивает макет и инициализирует ссылку на `graphS`.
*   **Когда вызывается:** При создании Activity.

### Имя метода: `viewGraph`
*   **Тип метода:** `public`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `View view`
*   **Что делает метод:** 
    1. Увеличивает счетчик `i`.
    2. В зависимости от значения `i` вызывает `buildGraph` с различными вариациями базовых коэффициентов (например, отрицательное 'a', 'b' или 'c').
*   **Когда вызывается:** При нажатии пользователем кнопки "View Graph".

### Имя метода: `buildGraph`
*   **Тип метода:** `public`
*   **Возвращаемое значение:** `void`
*   **Параметры:** `int a, int b, int c`
*   **Что делает метод:** 
    1. Создает новую серию данных `LineGraphSeries`.
    2. Вычисляет $y = ax^2 + bx + c$ для $x$ от -20 с шагом `dx`.
    3. Добавляет вычисленные точки в серию.
    4. Добавляет серию на график.
    5. Устанавливает цвет линии в зависимости от текущего значения счетчика `i`.
*   **Когда вызывается:** Из метода `viewGraph`.

## 4. Жизненный цикл
*   **`onCreate()`:** Инициализация и привязка интерфейса.

## 5. Взаимодействие с интерфейсом (UI)
*   **Кнопка:** Запускает добавление нового графика при каждом нажатии.
*   **GraphView:** Накапливает и отображает все добавленные серии данных.

## 6. Изаимодействие с другими компонентами
*   Запускается через Intent из `MainActivity`.

## 7. Общая логика класса
Каждое нажатие кнопки «добавляет» новую параболу к существующему графику. Класс использует счетчик, чтобы решить, какую математическую вариацию показать следующей, позволяя пользователю сравнить четыре различных состояния квадратичного уравнения.

## 8. Упрощенное объяснение
Представьте, что у вас есть прозрачный лист бумаги поверх сетки. Каждый раз, когда вы нажимаете кнопку, приложение рисует новую цветную кривую. К четвертому нажатию у вас будет четыре разные кривые на одном листе, что позволяет легко увидеть, как изменение чисел в формуле перемещает кривую.

**Аналогия:** Это как примерять четыре футболки разного цвета одну поверх другой, чтобы увидеть, какая лучше сидит или чем они отличаются.
